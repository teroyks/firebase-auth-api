# Installation

```shell
firebase login
firebase init
```

- select features
  - functions
  - hosting
- functions setup
  - use TypeScript
  - use TSLint
- hosting setup
  - configure as single-page app

```shell
npm install express
npm install --save-dev @types/express
```

- create a hosting rewrite
- initialize an express.js app in index
- add hosting to `npm run serve`
